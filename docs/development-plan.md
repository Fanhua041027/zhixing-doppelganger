# 开发计划

## 第一阶段：最小工作台

- 替换模板页，形成知行分身首页。
- 保持第一版本地化、确定性，保证演示稳定。
- 提供目标输入、意图识别、任务卡片和画像信号。
- 确认项目能在 DevEco Studio 26.0.0 Beta1 中打开和构建。

## 第二阶段：智能体主链路

- 将意图识别放入独立服务层。
- 增加任务拆解模型和执行状态。
- 保存任务历史。
- 为不明确目标提供兜底回复。
- 从第一版演示开始展示评测指标。

## 第三阶段：个人分身数据

- 建模用户偏好、学习习惯、记忆和冲突记录。
- 增加便于导入的结构化数据文件。
- 预留检索增强接口，避免界面绑定具体模型供应商。

## 第四阶段：鸿蒙原生能力

- 为日程和提醒场景接入系统意图。
- 原型化手机和平板间的任务展示。
- 增加权限和能力检查，并提供降级方案。

## 第五阶段：比赛交付

- 准备稳定演示路径。
- 编写项目摘要、创新点和架构说明。
- 增加意图准确率、任务完成度和个性化程度等评测指标。
