# 提交检查清单

## 代码

- 首页不再是模板内容。
- 首页提供目标模板，便于稳定演示不同学习场景。
- 对话、任务、画像、设置页面已注册到 `main_pages.json`。
- 评审展示页已注册到 `main_pages.json`。
- 提交总览页已注册到页面配置。
- 智能体规划由 `AgentService` 实现。
- 对话由 `ConversationService` 实现。
- 个人数据通过仓储层结构化。
- 系统能力调用隔离在 `HarmonyIntentService`。
